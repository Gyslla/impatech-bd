MC Ênfase em Ciência da Computação

Banco de Dados

54 horas Ano 2 · Período 2.3 Código-MC 24

Introdução a sistemas de banco de dados; projeto de banco de dados; modelo de dados relacional; modelo conceitual de entidades e relacionamentos, e UML; dependências funcionais e normalização; linguagens de definição e de manipulação de dados; álgebra relacional e SQL; consultas em dados relacionais; restrições de integridade; recursos avançados em SQL; organização física de bancos de dados: técnicas de armazenamento e indexação; Big data: definição, características e exemplos de domínios de aplicação; evolução dos requisitos de gerenciamento de dados e dos modelos de bancos de dados; bancos de dados NoSQL; bancos de dados NewSQL; Bancos de dados em memória. Organização física de bancos de dados: técnicas de armazenamento e indexação; Big data: definição, características e exemplos de domínios de aplicação; evolução dos requisitos de gerenciamento de dados e dos modelos de bancos de dados; bancos de dados NoSQL; bancos de dados NewSQL.
